## Oor hierdie repo

Statiese webwerf — geen bou-stap nodig nie. `index.html` bevat al die HTML, CSS
en JavaScript; die foto's lê in `Images/`.

| Lêer | Beskrywing |
| --- | --- |
| `index.html` | Die hele webwerf (HTML + CSS + JS) |
| `landing-page.jpg` | Vaste agtergrondfoto |
| `Logo.jpg` | Logo |
| `Images/` | Galery- en diensfoto's |

Foto's is vir die web verklein (maks. 1600px) en saamgepers.

### Om iets te verander

Wysig `index.html` en stoot na `main` — GitHub Pages werk outomaties by.
